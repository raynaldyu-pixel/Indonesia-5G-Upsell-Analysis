# 🇮🇩 Indonesia 5G Upsell Analysis

[![Streamlit App](https://img.shields.io/badge/Streamlit-App-FF4B4B?style=flat&logo=streamlit&logoColor=white)](https://indonesian-5g-upsell-analysis.streamlit.app/)

An interactive data analytics project focused on predicting and analyzing consumer behavior for 5G network upgrades in the Indonesian telecommunications sector. This project leverages machine learning to identify key drivers for upsells and provides an interactive dashboard for business insights.

🚀 **[Access the Live Web Application Here](https://indonesian-5g-upsell-analysis.streamlit.app/)**

---

## 📌 Project Overview
As 5G infrastructure expands across Indonesia, telecom operators face the challenge of efficiently converting existing 4G users to premium 5G plans. This repository contains the end-to-end data pipeline, predictive modeling, and insights used to identify high-potential customers for 5G upsells.

### Key Objectives:
* **Predictive Modeling:** Build robust machine learning models to identify customers with the highest probability of upgrading.
* **Feature Importance:** Analyze behavioral and demographic indicators (e.g., data usage patterns, device compatibility, regional availability) that drive 5G adoption.
* **Business Strategy:** Translate model outputs into actionable marketing and upsell recommendations.

---

## 🛠️ Tech Stack & Architecture
* **Frontend Dashboard:** Streamlit
* **Languages:** Python (Pandas, NumPy)
* **Machine Learning:** Scikit-learn / XGBoost / Optuna
* **Data Visualization:** Plotly / Seaborn / Matplotlib

---

## 📊 Dashboard Features
* **User Segmentation:** Deep dive into customer profiles across different regions in Indonesia.
* **Predictive Sandbox:** Input hypothetical customer metrics to see real-time 5G upgrade probability scores.
* **Interactive Charts:** Dynamic visualizations illustrating usage trends and churn vs. upgrade correlations.
